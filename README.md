# nerdps1
Nerd prompt for bash/ksh/zsh

## activate the prompt

```shell
$ . <(curl -s https://raw.githubusercontent.com/joknarf/nerdps1/main/nerdps1)
or
$ . <(curl -sL https://shorturl.at/cjtzE)
```
